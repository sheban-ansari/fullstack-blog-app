# Full Stack Blog App

A modern Full Stack Blog Application built using the MERN Stack (MongoDB, Express, React, Node.js).
This project allows users to create, read, update, and delete blog posts with a clean UI and REST API integration.

---

##  Live Features

- Create Blog Posts
- Edit Existing Blogs
- Delete Blogs
- Read All Blogs (Cards UI)
- REST API Integration
- Modern UI with Tailwind CSS
- Secure Environment Variables (.env)

---

##  Tech Stack

### Frontend:

* React (Vite)
* Tailwind CSS
* Axios / Fetch API
* CSS (Modern UI)

### Backend:

* Node.js
* Express.js
* MongoDB (Mongoose)
* CORS
* Dotenv
* Nodemon
